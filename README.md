# GoalSetter_MernApp
This is the goalsetter app with complete CRUD functionality. 
## Usage
Rename the .envexample to .env and add your MONGO_URI
### Install dependencies
```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```
### Run Server
```
npm run server
```
